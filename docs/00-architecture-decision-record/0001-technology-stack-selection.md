# ADR 0001 - Technology Stack Selection

| **Title** |  Technology Stack Selection.|
| :--- | :--- |
| **Context** | Need a stable, modern, and high-performance environment to develop a NFL project. |
| **Decision** | Use Java 21 (LTS) and Maven as the project management tool. |
| **Status** | `Accepted` |
| **Consequences** | Access to modern Java features (Virtual Threads, Records). Standardized dependency management via Maven Central. Portable to multiple platforms (desktop, web backend, potential Kotlin multiplatform in the future).
